# my-work (example)

> This is an example project. See `projects/README.md` for how to remove it and start your own.

## Overview

Placeholder for a day-job project. In a real workspace this is where you'd track tickets, architecture notes, meeting minutes, and anything else you want your AI agents to have context on.

## Typical layout

```
my-work/
├── PROJECT.md        # this file — required for discovery
├── notes/            # free-form markdown notes
├── plans/            # design docs and implementation plans
└── progress/         # per-session progress logs (YYYY-MM-DD-slug.md)
```
